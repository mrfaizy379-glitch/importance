# Importance Website

An interactive web application built using modern frontend technologies to deliver a smooth and engaging user experience. The project focuses on clean design, responsive layout, and subtle animations to enhance usability and visual appeal.

---

## Tech Stack

* **Next.js** – React framework for building fast and scalable applications
* **Tailwind CSS** – Utility-first CSS framework for responsive design
* **Framer Motion** – Animation library for smooth transitions and effects

---

## Features

* Responsive and modern UI
* Smooth animations and transitions
* Optimized performance with Next.js
* Clean and maintainable code structure

---

## Local Setup

To run this project locally, follow these steps:

```bash
# Clone the repository
git clone <repository-url>

# Navigate into the folder
cd importance-site

# Install dependencies
npm install

# Start the development server
npm run dev
```

After running the development server, open the following URL in your browser:

```
http://localhost:3000
```
